## Recording Tips

### Setup
- Use **Loom** (loom.com) — free, records screen + your face in corner
- Record at 1920×1080, 30fps minimum
- Use a headset or AirPods — laptop mic sounds bad
- Record in a quiet room, one take is fine

### Before You Hit Record
- Log in to Vigmis in a separate browser so the dashboard has real data showing
- Open all pages in tabs beforehand so navigation looks smooth
- Use Chrome — cleanest screen recording

### For TikTok API Application
- The 90-second version is what TikTok reviewers actually watch
- Make sure the TikTok OAuth connection screen is visible
- Make sure you show content being *published* to TikTok (Scene 4)
- Submit in MP4 format, max 200MB

### After Recording
- Loom auto-generates a link you can share
- Download as MP4 for TikTok submission
- Caption optional for TikTok application, required for marketing version
